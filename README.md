# MOC-RVQ

This repository contains the official PyTorch implementation for the paper  
**[MOC-RVQ: Multilevel Codebook-Assisted Digital Generative Semantic Communication (GLOBECOM 24)](https://arxiv.org/abs/2401.01272)**.
